产生地点.py 用于生成上海市范围内等距离的中心点，后面爬取时以该点为中心进行周边停车场数据的爬取

爬取路内停车位.py 用于爬取路内停车位的相应的信息

爬取路外停车位.py 用于爬取路外停车场的相应的信息

爬取图片及详细信息.py 用于根据车场id进行额外的信息爬取

-----------------------------------------------------------------------------------------
获取的地点.txt 为生成的中心点的经纬度信息

Parking_type1.txt 为路外停车场的id集合

Parking_type2.txt 为路内停车位的id集合

自动脚本.bat 用于在Windows计算机上实现自动爬取
